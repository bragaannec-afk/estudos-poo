# Estudos POO

Este repositório contém materiais e exemplos de estudo sobre Programação Orientada a Objetos (POO). O objetivo é fornecer uma base sólida para entender e aplicar conceitos fundamentais da POO, como classes, objetos, herança, polimorfismo, encapsulamento e abstração.

## Estrutura do Projeto

- **Exemplos/**: Contém códigos de exemplo em linguagens como Java, Python ou C#, demonstrando conceitos de POO.
- **Exercícios/**: Exercícios práticos para praticar os conceitos aprendidos.
- **Notas/**: Anotações e resumos teóricos sobre POO.

## Pré-requisitos

- Conhecimento básico de programação.
- Ambiente de desenvolvimento configurado (ex.: JDK para Java, Python instalado, etc.).

## Como Usar

1. Clone o repositório: `git clone https://github.com/seu-usuario/estudos-poo.git`
2. Navegue para a pasta: `cd estudos-poo`
3. Execute os exemplos conforme as instruções em cada arquivo.
